# SnapTrans 螢幕截圖翻譯工具

這是一個輕量級的桌面應用程式，使用 Python 與 PyQt5 開發。

## 功能特色
1. **DPI 感知**：自動適應 Windows 高解析度縮放，保證截圖座標無誤差。
2. **截圖選取**：全螢幕半透明遮罩，滑鼠拖曳框選欲翻譯範圍。
3. **OCR 整合**：採用 `pytesseract` 自動辨識圖片內的英文。
4. **文字前處理**：內建 Regex 清洗，自動刪除換行與連字符號，還原完整段落。
5. **非同步翻譯**：透過 `deep-translator` 呼叫線上翻譯，背景執行維持畫面不卡頓。
6. **結果雙語顯示**：專屬小視窗呈現中英對照，支援一鍵複製。

## 環境準備與安裝

1. 請確保您的電腦上已安裝 [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki)，並將其加入至系統的環境變數 (PATH)。
   * 如果沒有加入系統變數，請在 `core.py` 中解除註解指定路徑。
2. 開啟終端機 (命令提示字元或 PowerShell)，進入 `SnapTrans` 資料夾。
3. 執行安裝指令以安裝必要的套件：
   ```bash
   pip install -r requirements.txt
   ```

## 如何使用

執行主程式啟動工具：
```bash
python main.py
```
啟動後，滑鼠游標會變成十字。請在螢幕上拖曳選取想要翻譯的英文字。
放開滑鼠後會自動進行 OCR 辨識與連線翻譯，完成後會有彈窗顯示「英文原文」以及「中文翻譯」，可以輕易將內容複製出來。
按 ESC 可以取消選取並關閉程式。
